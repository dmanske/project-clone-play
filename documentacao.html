Sistema de Gerenciamento de Caravanas

Visão Geral

O Sistema de Gerenciamento de Caravanas é uma aplicação web desenvolvida para gerenciar caravanas de torcedores do Flamengo, permitindo o controle de viagens, ônibus, passageiros e pagamentos.

Funcionalidades Implementadas

  • Gerenciamento de Viagens Implementado
    • Cadastro de novas viagens
    • Edição de viagens existentes
    • Visualização de detalhes da viagem
    • Controle de status da viagem
  • Gerenciamento de Ônibus Implementado
    • Adição de ônibus às viagens
    • Remoção de ônibus (com validação de passageiros)
    • Controle de capacidade dos ônibus
  • Gerenciamento de Passageiros Implementado
    • Cadastro de novos passageiros
    • Associação de passageiros a ônibus específicos
    • Controle de status de pagamento
    • Notificação via WhatsApp (simulada)
  • Interface do Usuário Implementado
    • Dashboard com visão geral
    • Navegação intuitiva
    • Formulários com validação
    • Feedback visual de ações
